# 目标

构建一个简洁易用的番茄钟应用，帮助用户通过 25 分钟专注 + 5 分钟休息的循环节奏提升工作效率。

# 需求

- 提供 25 分钟专注计时模式，倒计时归零后自动切换至休息阶段
- 提供 5 分钟休息计时模式，倒计时归零后自动切换至专注阶段
- 支持手动启动、暂停、继续计时
- 支持手动跳过当前阶段（直接切换到下一阶段）
- 支持重置当前阶段计时器
- 计时结束时发出声音或视觉提醒，告知用户阶段切换
- 页面实时显示当前剩余时间（分:秒格式）
- 页面清晰标识当前所处阶段（专注 / 休息）
- 记录并展示当前会话已完成的番茄钟数量

# 验收标准

- Given 用户打开应用，When 页面加载完成，Then 显示 25:00 倒计时及"专注"阶段标识，按钮状态为"开始"
- Given 用户点击开始，When 计时运行中，Then 每秒更新剩余时间，按钮切换为"暂停"
- Given 用户点击暂停，When 计时暂停，Then 时间停止变化，按钮切换为"继续"
- Given 专注阶段倒计时归零，When 时间到达 00:00，Then 触发提醒音/视觉提示，自动切换至 5 分钟休息阶段并开始倒计时
- Given 休息阶段倒计时归零，When 时间到达 00:00，Then 触发提醒音/视觉提示，自动切换至下一轮 25 分钟专注阶段，已完成番茄钟数 +1
- Given 用户点击跳过，When 处于任意阶段，Then 立即切换至下一阶段并重置对应时间
- Given 用户点击重置，When 处于任意阶段，Then 当前阶段时间恢复初始值，计时停止
